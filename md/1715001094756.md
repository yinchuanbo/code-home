```html
<div>
  <span id="rangeValue">50</span>
  <input
    class="range"
    type="range"
    value="50"
    min="0"
    max="100"
    onChange="rangeSlide(this.value)"
    onmousemove="rangeSlide(this.value)"
  />
</div>
<script>
  function rangeSlide(value) {
    document.getElementById('rangeValue').innerHTML = value;
  }
</script>
```
